# JarvisAi Unity Product Contract

## What it is

JarvisAi Unity is a fast desktop assistant with one main focus:

- write a request
- say a request
- execute useful actions quickly

## Core screens

1. Chat
2. Voice
3. Apps
4. Settings
5. Registration

## Keep

- chat-first home
- manual mic
- wake word
- command palette
- quick local actions
- multi-command execution
- Groq-backed smart replies
- clean first-run registration
- updater

## Remove

- diagnostics as a user-facing section
- system shell as a user-facing section
- old control center
- large debug surfaces
- permanent `Нубик` block on the main screen

## Product tone

- dark, premium, calm
- quick, direct, not overloaded
- glowing identity without turning into a sci-fi dashboard

## Registration

Registration stays in the product as a stylish onboarding screen with:

- Groq API Key
- Telegram User ID
- Telegram Bot Token

Helper links are shown directly under the matching fields.
