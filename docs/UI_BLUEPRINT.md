# UI Blueprint

## Navigation

- `Chat`
- `Voice`
- `Apps`
- `Settings`

Settings stays pinned low in the sidebar.

## Chat screen

- top title and compact assistant state
- queue ribbon for multi-command execution
- message timeline
- quick action strip
- composer with send and mic

## Voice screen

- wake word switch
- voice mode selector
- command style selector
- microphone selector
- wake-word test
- runtime summary card

## Apps screen

- launch catalog
- aliases
- enable/disable
- usage hints

## Settings screen

- appearance
- startup
- privacy
- AI provider/model
- updater info
- compact `Нубик` guide as a settings navigator only

## Registration screen

- centered onboarding card
- short explanation
- 3 inputs
- helper links below inputs
- continue button
- configure later action
